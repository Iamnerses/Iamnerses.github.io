# Iamnerses Portfolio & Project Catalogue

A modern, high-performance, and fully customizable personal portfolio website built for GitHub Pages. This site serves as a central hub for showcasing games, mobile apps, web programs, and blog posts.

## Live Demo
Visit the live site here: [https://iamnerses.github.io](https://iamnerses.github.io)

## Features

-   **Dynamic Content Engine**: All data (profile, blog, games, apps) is controlled by a single `data.json` file.
-   **Integrated CMS**: A built-in `/edit.html` panel allows for easy content management without touching code.
-   **Game Catalogue**: Seamlessly play hosted games (like Bardle and Broken Phone) via interactive iframe modals.
-   **Theme System**: Includes Dark (Default), Light, and Cyberpunk themes with persistent user preferences.
-   **Responsive & Fast**: Built with pure HTML, CSS, and Vanilla JS for near-instant load times and 100% SEO friendliness.
-   **Markdown Blog**: Write blog posts in Markdown and have them rendered beautifully in-app.

## Tech Stack

-   **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
-   **Markdown Rendering**: [Marked.js](https://marked.js.org/)
-   **Fonts**: Inter & Space Grotesk (Google Fonts)
-   **Hosting**: GitHub Pages

## Project Structure

```text
.
├── index.html          # Main portfolio entry point
├── edit.html           # Content management panel
├── style.css           # Global design system & animations
├── data.json           # Content database (JSON)
└── assets/             # Images, avatars, and static media
```
